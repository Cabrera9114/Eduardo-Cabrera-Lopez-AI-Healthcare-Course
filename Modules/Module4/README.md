# Module 4: AI-Based Diagnostic Tools

## 📌 Overview
This module focused on AI-based diagnostic tools and how machine learning can be used to improve disease detection and clinical accuracy.

## 🧠 What I Learned
In this module, I learned how AI systems use deep learning and computer vision to analyze medical images and assist in diagnosing diseases such as skin cancer. I also learned how to evaluate diagnostic tools using performance metrics like sensitivity and specificity.

## ⚙️ Work Completed
For this assignment, I analyzed a diagnostic study from Stanford Medicine involving an AI system designed to detect skin cancer from images of skin lesions.

The AI system:
- Uses deep learning and image recognition
- Compares images against large datasets
- Identifies patterns associated with cancer
- Assists clinicians in making diagnostic decisions

## 📊 Key Insights
The study showed that using AI assistance improved diagnostic performance:

- **Sensitivity increased** (better detection of actual cancer cases)
- **Specificity increased** (fewer false positives)

## ⚠️ Challenges
Despite its benefits, the system still has limitations:
- False positives (incorrectly identifying cancer)
- False negatives (missing actual cancer cases)
- Need for human oversight

This highlights the importance of combining AI with clinical expertise.

## ✅ Conclusion
This module demonstrated how AI-based diagnostic tools can significantly improve healthcare outcomes. While not perfect, these systems provide valuable support to clinicians and enhance decision-making.
