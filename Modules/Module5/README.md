# Module 5: AI Risk Modeling and Prognosis

## 📌 Overview
This module focused on AI-based prognostic models and how machine learning can be used to predict patient outcomes and disease risk.

## 🧠 What I Learned
In this module, I learned how machine learning models can analyze patient data to predict future health risks, such as heart disease. I also explored how these models differ from traditional statistical methods and how they are evaluated using performance metrics.

## ⚙️ Work Completed
For this assignment, I analyzed a machine learning-based prognostic model used to predict cardiovascular risk, including heart attacks and strokes.

The model uses:
- Age
- Blood pressure
- Cholesterol levels
- Smoking status
- Diabetes status
- Body Mass Index (BMI)

It analyzes electronic health record (EHR) data to detect patterns and estimate a patient’s risk level.

## 📊 Key Insights
The study showed that the machine learning model performed better than traditional risk models.

- It used **AUC (Area Under the Curve)** to measure performance  
- It demonstrated better ability to distinguish between high-risk and low-risk patients

## ⚠️ Challenges
Some limitations of the model include:
- Lack of transparency (black-box nature of AI)
- Potential bias if training data is not diverse
- Need for continuous validation across populations

These challenges show the importance of ethical AI use in healthcare.

## ✅ Conclusion
This module demonstrated how AI can be used for risk prediction and prognosis in healthcare. These models can improve early intervention and patient outcomes when used responsibly alongside clinical expertise.
