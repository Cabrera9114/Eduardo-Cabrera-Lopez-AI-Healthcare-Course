# Module 9: Reinforcement Learning in Healthcare

## 📌 Overview
This module focused on Reinforcement Learning (RL) and how it can be applied in healthcare to support continuous decision-making and personalized treatment strategies.

## 🧠 What I Learned
In this module, I learned how reinforcement learning works by allowing systems to learn from actions and feedback over time. I also explored how RL can be used in healthcare scenarios that require ongoing adjustments, such as medication management.

## ⚙️ Work Completed
For this assignment, I designed a reinforcement learning scenario for managing medication dosage in patients with high blood pressure.

The RL system includes:

- **State:** Patient condition (blood pressure, heart rate, age, weight, medication dosage, side effects)
- **Actions:** Increase, decrease, or maintain medication dosage
- **Reward:** Positive outcomes for improved health and no side effects; negative outcomes for worsening conditions

## 📊 Key Insights
This assignment showed that reinforcement learning can:
- Adapt treatment plans over time  
- Personalize care for individual patients  
- Improve long-term decision-making  
- Support dynamic healthcare environments

## ⚠️ Challenges
There are important challenges when applying RL in healthcare:
- Safety risks during early learning stages  
- Difficulty designing accurate reward systems  
- Need for human oversight  
- Requirement for extensive testing before real-world use  

These challenges highlight the importance of cautious implementation.

## ✅ Conclusion
This module demonstrated the potential of reinforcement learning in healthcare for improving patient outcomes through adaptive and personalized treatment strategies. RL should be used as a support tool alongside healthcare professionals.
